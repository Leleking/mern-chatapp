# ChatApp by Simeon Nortey

[![N|Solid](http://simeon.symliq.com/img/simeon.png)](https://nodesource.com/products/nsolid)



ChatApp by Simeon Nortey. This application is a chatapp created using react, node, express for API, mongodb and Auth0 for authentication.


 




### Installation



Install the dependencies and devDependencies and start the server.

```sh
install docker

$ docker-compose up
```

View configuration files

```sh
$ touch ./client/configuration.json
$ touch ./server/configuration.json
```
